# Codex Attempt

This repository provides a very small example website that can be run locally using Node.js and Express.

## Setup

1. Install dependencies:
   ```bash
   npm install express
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open your browser at <http://localhost:3000> to see the site.

Feel free to modify the HTML, CSS, or JavaScript in the `public/` directory.
